# 产品形态

DataLuminary 的产品形态不是单一报表工具，而是一套可以面向不同客户、行业和部署规模组合交付的数据洞察平台。

## 1. 自助 BI 平台

面向业务团队、运营团队和管理者，提供从数据源到仪表盘的低代码分析体验：

- 数据源连接、数据集建模、指标维度配置。
- 拖拽式图表和仪表盘搭建。
- 多空间协作、权限控制、发布分享和访问统计。
- 管理驾驶舱、经营报表、业务监控和专题分析。

## 2. AI 数据分析工作台

面向希望降低分析门槛的组织，DataLuminary 提供人机协作的数据分析体验：

- 用自然语言描述问题，自动生成数据查询、图表和洞察。
- AI 分步骤协助完成数据集、指标、图表、布局和报告。
- 生成结果沉淀为平台原生资源，可编辑、可审计、可复用。
- 支持异常解释、趋势归因、指标波动说明和后续分析建议。

## 3. 企业数据资产与指标中台

面向需要统一口径的大中型组织，DataLuminary 可以承载轻量数据语义层：

- 指标、维度、字段字典、计算口径和展示说明。
- 数据血缘、数据质量提示和跨部门指标治理。
- 权限、空间、版本、发布、订阅和审计。
- 为管理层、业务线、数据团队提供同源分析资产。

## 4. 大屏与管理驾驶舱平台

面向展示、监控和经营决策场景，DataLuminary 支持多形态可视化表达：

- PC 网格仪表盘、自由布局大屏、移动列表、报表卡片。
- 生产监控、IoT 设备看板、销售驾驶舱、财务分析、运维审计。
- 图表联动、下钻、变量筛选和时间范围控制。
- 可嵌入第三方系统，也可独立作为展示门户。

## 5. 嵌入式分析引擎

面向 SaaS、行业系统和内部业务系统，DataLuminary 可以作为嵌入式 BI 能力：

- 将图表、仪表盘、报表和洞察模块嵌入现有业务页面。
- 通过权限、Token、租户和资源隔离适配多客户场景。
- 通过插件和 Schema 保持配置可迁移、可复用、可版本化。

## 6. 插件与行业模板生态

面向开发者、社区和合作伙伴，DataLuminary 提供可持续扩展的插件生态：

- 数据源插件：数据库、文件、API、IoT、日志、Agent 和业务系统。
- 图表插件：行业图表、地图、指标卡、多媒体、自定义渲染。
- 面板插件：网格、大屏、移动、报表、嵌入式组件。
- AI 插件：问数、配图、报告、异常检测、数据质量检查。
- 行业模板：制造、教育、运维、零售、金融、Agent 平台等场景方案。

## 7. 私有化 AI 数据工作台

面向重视数据安全、合规和网络隔离的客户，DataLuminary 可以在私有环境中交付：

- 内网部署、私有数据库、私有模型和本地权限体系。
- 适配不同数据规模，从单部门试点到企业级多空间使用。
- 支持备份恢复、审计、可观测性和运维交付流程。
- 通过插件快速适配客户现有数据系统和业务流程。

## 产品边界

DataLuminary 的边界不是“做更多页面”，而是围绕数据洞察形成稳定平台能力：

```text
连接数据 -> 治理语义 -> 生成分析 -> 表达洞察 -> 协作决策 -> 生态扩展
```

在这个边界内，产品可以服务普通 BI 用户，也可以服务开发者、交付团队、行业伙伴和投资者关注的平台化增长。
