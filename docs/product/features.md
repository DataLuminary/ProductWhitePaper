# 当前版本能力（约 v0.8+）

以下为当前已交付、可在产品中使用的核心能力。

## 账号与权限

- 用户注册、登录（含 SSO 对接能力）
- 基于角色与资源的访问控制
- 工作空间（Space）下的资源隔离

## 数据接入

- **数据源管理**：创建、编辑、测试连接
- **已支持数据源类型**：
  - MySQL
  - Excel 文件导入
  - ClickHouse
- 数据源配置界面与连接管理

## 数据集

- 基于数据源创建数据集
- 配置 **指标** 与 **维度** 字段
- 数据预览与查询，供图表消费

## 图表与可视化

- **NoCode 配图**：通过配置界面选择图表类型与数据映射
- **内置图表类型**（24+），例如：
  - 折线、柱状、面积及堆叠/组合变体
  - 饼图、环形、玫瑰、雷达
  - 表格、散点、漏斗、词云、地图
  - 指标卡、仪表盘、水球、富文本、图片、视频等
- 图表作为资源管理，可挂载到仪表盘

## 仪表盘

- 仪表盘创建、编辑、发布与管理
- **网格布局（grid-layout）** 拖拽编排
- 目录分类、版本记录、收藏与访问统计
- 基础 **交互组件**：输入、下拉、单选、时间选择等，用于筛选图表

## 尚未包含（见「未来版本」）

- 字典（Dictionary）完整配置流程
- 列表布局、自由布局（大屏）作为正式面板插件
- PostgreSQL、SQL Server、通用 API 数据源
- AI 自动配图与洞察报告
