# DataLuminary 产品说明

DataLuminary 是一款开源、插件化的数据可视化与洞察平台，帮助团队以更低代码成本完成从数据接入到仪表盘分析的全流程。

## 文档说明

本站 **产品章节** 面向使用者与业务方，说明：

- [当前版本能力](./features.md)：现在能做什么
- [未来版本规划](./upcoming.md)：后续将增加什么
- [产品愿景](./vision.md)：长期方向

开发与发版规划（里程碑、版本节奏、治理规范）维护在 MetaRepo 的 [`plan/`](https://github.com/DataLuminary/DataLuminary-Platform/tree/main/plan) 目录，不在此重复展开。

## 核心能力一览

- 多数据源接入与数据集建模
- 丰富的图表插件与 NoCode 配图
- 仪表盘管理与网格布局
- 权限与空间协作基础能力
