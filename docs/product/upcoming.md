# 未来版本规划

规划会随开发节奏调整，以下为当前方向（非承诺日期）。

## 近期（v0.9–v0.10）

- 平台稳定性与文档体验优化
- **更多布局**：列表布局、自由布局（适合移动端与大屏）
- **更多数据源**：PostgreSQL、SQL Server、API 类连接器
- **字典**：字段字典配置与图表展示增强
- 图表 **联动、下钻** 等交互配置增强

## 中期（v0.11+）

- **AI 配图**：用自然语言描述需求，生成图表配置草案并可手工调整
- **AI 仪表盘**：根据分析目标推荐布局与图表组合
- **数据洞察报告**：基于数据集生成可解释的洞察摘要

## 长期（v1.0 及以后）

- 面向开源社区的完整贡献与协作流程
- 企业级：审计、性能、可观测性增强
- 更开放的插件生态与第三方对接（如 Grafana 面板迁移深化）

## 说明

- 详细里程碑与版本节奏见仓库 [`plan/`](https://github.com/DataLuminary/DataLuminary-Platform/tree/main/plan)（开发者文档）。
- 欢迎通过 Issue 讨论优先级。
