# 产品愿景

## 我们要解决的问题

许多团队有数据，但缺少轻量、可扩展、可协作的 BI 工具：接入成本高、配图门槛高、定制开发重。

## 愿景

构建一个 **开源、插件化、可解释** 的智能 BI 平台，让业务与研发都能以更低成本完成：

**数据接入 → 数据集建模 → 可视化 → 洞察决策**

## 设计原则

- **插件化**：图表、面板、数据源、交互皆可扩展
- **LessCode / NoCode**：拖拽与表单配置为主，必要时可深入 SQL/高级配置
- **Headless BI 思路**：数据集统一承载指标与维度，图表专注呈现
- **全栈 TypeScript**：降低协作与维护成本，便于 AI 辅助开发（Vibe Code）
- **可演进**：规格驱动开发，版本与能力对用户透明

## 长期方向

- 从「能看图」到「能解释数据」：AI 辅助配图与洞察
- 从「单人可用」到「社区可贡献」：开源协作与插件市场形态
- 从「单场景」到「多端多布局」：PC 分析、移动看板、大屏展示

## 当前阶段

核心 BI 闭环已可用；正在完善工程治理与文档体系，并规划插件扩展与 AI 能力。
