# DataLuminary 与 LuminaryWorks AI 生态

> 用户向说明 · 工程规格见 [spec/ecosystem.md](../../spec/ecosystem.md)

## 本产品是什么

**DataLuminary** 是 AI 数据洞察平台：数据集、DataTalk 可视化大屏、权限与 AI 辅助分析。可**单独采购与部署**，服务金融、制造、运营等 BI 场景。

## 在 LuminaryWorks 五项目中的位置

LuminaryWorks 是五个**可独立商业化**、又可组合的开源 AI 产品家族：

| 产品 | 角色 |
|------|------|
| **DataLuminary（本产品）** | **看见** — 把数据变成可决策的图表与大屏 |
| VibeEdu | **学** — AI 编程与 IoT 实验教育 |
| LuminaryIoTChain | **连** — 开源 IoT PaaS，设备上云 |
| VistaRemote | **控** — WebRTC 远程运维 |
| VibeAgent | **赚** — AI Agent 链上交易市场 |

```text
数据 / 遥测 / 录制 ──► DataLuminary DataTalk ──► 决策与展示
```

## 常见组合场景

| 场景 | 组合 |
|------|------|
| 制造企业 | LuminaryIoTChain + **DataLuminary** 设备监控大屏 |
| 运维团队 | VistaRemote + **DataLuminary** 审计与效率报表 |
| 纯 BI 客户 | **仅 DataLuminary** |

## 统一体验（可选）

接入 LuminaryWorks 共享登录后，同一账号可跳转兄弟产品；未接入时仍使用 DataLuminary 本地账号体系。
